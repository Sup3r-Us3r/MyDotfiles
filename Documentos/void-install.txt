VOID INSTALL

Instalar a .iso

Conectar a internet
$ ip addr (pegar a interface de rede)
$ sudo iw dev wlp2s0 scan | grep -i ssid
$ wpa_passphrase {ssid} {password} | sudo tee -a /etc/wpa_supplicant/wpa_supplicant.conf
$ cat /etc/wpa_supplicant/wpa_supplicant.conf
$ sudo wpa_supplicant -B -D wext -i wlp2s0 -c /etc/wpa_supplicant/wpa_supplicant.conf 
$ sudo iw wlp2s0 link

Atualizar sistema
$ sudo xbps-install -Suv (exe
cutar 2 vezes)

Remover serviços inúteis
$ sudo rm -fv /var/service/dhcpcd
$ sudo rm -fv /var/service/{agetty-tty2,agetty-tty3,agetty-tty4,agetty-tty5,agetty-tty6}

Adicionar serviços
$ sudo xbps-install -S elogind polkit alsa-utils ConsoleKit2
$ sudo ln -srf /etc/sv/{dbus,polkitd,elogind,cgmanager,consolekit} /var/service
$ sudo ln -s /etc/sv/alsa /var/service/alsa

Criar um usuário caso não exista
$ useradd -m -s /bin/bash -U -G wheel,users,audio,video,cdrom,input,storage,power ghost
$ passwd ghost
$ sudo visudo (descomentar a linha: # %wheel ALL=(ALL) ALL)

Instalar Xorg
$ sudo xbps-install xorg-minimal xinit xf68-video-vesa
$ echo "exec ck-launch-session dbus-launch --sh-syntax --exit-with-x11 i3" > ~/.xinitrc

Adicionar repositório multilib
$ sudo xbps-query -Rs void-repo
$ sudo xbps-install void-repo-nonfree
$ sudo xbps-install -Suv

Instalar pacote base
$ sudo xbps-install base-devel

Remover kernel antigo
$ uname -r
$ vkpurge list
$ sudo vkpurge rm 4.17.5_1 (versão do kernel) * Irá atualizar o grub automaticamente



Xbps-src - Instalar google chrome
$ sudo xbps-install -S base-devel git xtools
$ git clone https://github.com/void-linux/void-packages.git
$ cd void-packages/
$ ./xbps-src binary-bootstrap
$ echo XBPS_ALLOW_RESTRICTED=yes >> etc/conf
$ ./xbps-src pkg google-chrome
$ sudo xbps-install --repository=~/void-packages/hostdir/binpkgs/nonfree google-chrome












Iniciar serviços
$ sudo sv start service

Parar serviços
$ sudo sv stop service

Pesquisar algum pacote
$ sudo xbps-query -Rs pkg

Listar pacotes instalados
$ xbps-query -l | wc -l
