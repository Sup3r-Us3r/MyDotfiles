### `Monsterwm`
```sh
$ sudo pacman -S git conky
$ cd ~/.config/
$ git clone https://github.com/c00kiemon5ter/monsterwm.git
$ cd monsterwm
$ cp config.def.h config.h
$ nano config.h
$ make
$ sudo make clean install
/// To use the bar just run: conky -c ~/.config/monsterwm/bar
```
